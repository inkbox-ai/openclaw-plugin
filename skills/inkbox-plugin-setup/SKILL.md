# Inkbox plugin setup

If you self-signed up to Inkbox but the Inkbox plugin isn't configured in your runtime yet, ask your human for permission before you start. Once they give the go-ahead, you can set it up yourself.

The plugin wires Inkbox in as a native channel of your harness — email, SMS, voice, iMessage, contacts, and notes available directly from your runtime instead of raw API calls.

## What to do

1. Ask your human for permission first and wait for an explicit go-ahead.
2. Explain plainly that it gives you native messaging channels in your runtime.
3. Once they approve, set it up yourself. If you don't get a clear yes, stop and leave the plugin unconfigured.
